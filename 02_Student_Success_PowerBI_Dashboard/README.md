# Student Success Analytics Dashboard | Power BI

## Overview

This project was completed as the final project for my Data Visualization Lab, with the objective of building a dashboard that goes beyond reporting metrics and supports data-driven decision making.

The focus was on transforming student performance data into actionable insights by identifying academic, demographic, and geographic trends that could help stakeholders better understand factors influencing student success.

Rather than simply visualizing available data, the dashboard was designed around key questions and potential actions that could be taken from the analysis.

---

## Objective

Educational institutions collect large amounts of student data, but identifying meaningful patterns requires translating that information into clear insights.

The goal of this project was to answer questions such as:

- Which factors are associated with stronger college academic performance?
- How do academic outcomes differ across student groups?
- Are there patterns that could help guide student support strategies?

---

## Dashboard Approach

The analysis explored:

### Academic Indicators

- Relationship between high school GPA and college GPA
- Standardized testing trends
- Academic performance differences among student groups

### Student Segmentation

- Athlete vs non-athlete comparisons
- Gender-based performance analysis
- Admission cohort trends

### Geographic Insights

- Student distribution by location
- Regional differences in academic outcomes

## Key Focus: Insight to Action

The dashboard was designed to help users move from:

Data → Trends → Insights → Potential Actions

Examples:
- Identifying student groups that may benefit from additional academic support
- Understanding characteristics associated with stronger academic outcomes
- Recognizing trends across incoming student cohorts
