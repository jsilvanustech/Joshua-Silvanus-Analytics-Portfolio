# Student Success Analytics Dashboard


## Objective

Developed an interactive Power BI dashboard analyzing student 
academic outcomes to identify patterns across demographic, 
academic, and geographic factors.


## Analysis

The dashboard explores relationships between:

- High school academic performance and college GPA outcomes
- Standardized testing metrics and student performance
- Athlete and non-athlete academic differences
- Student performance across admission cohorts
- Geographic distribution patterns


## Approach

- Cleaned and structured student records for analysis
- Designed interactive dashboard pages for stakeholder exploration
- Created visual comparisons across student groups
- Developed geographic visualizations to identify regional patterns


## Dashboard Features

- GPA performance trends
- Academic profile comparisons
- Cohort analysis
- Demographic breakdowns
- Geographic mapping
