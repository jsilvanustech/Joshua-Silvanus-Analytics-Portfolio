## 1. Sunoco Gasoline Demand Forecasting: Predicting High-Demand Weeks with Machine Learning

### Objective
Developed a machine learning early-warning system to predict 
high-demand gasoline weeks and reduce costly supply over-allocation.

### Tools
Python | pandas | scikit-learn | XGBoost | Machine Learning

### Highlights
- Engineered macroeconomic, mobility, weather, and energy indicators
- Built classification models optimized around business costs
- Tuned XGBoost model to minimize false positives
- Achieved:
    - 79% precision
    - 73% high-demand identification
    - Only 3 false alerts annually
